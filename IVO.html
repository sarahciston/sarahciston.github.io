inner(voice)over

This audio sculpture and web app asks how we might 'rewrite' the inner critic using natural language processing. It creates a self-compassion database using an AI speech-to-text and text-to-speech pipeline built with GPT-3 text generation, Mozilla DeepSpeech voice recognition, voice synthesis tools, plus participant contributions.
